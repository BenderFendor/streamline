# non-goals.md -- Explicit scope boundaries

Streamline does NOT aim to be:

1. A streaming service -- it provides information and tracking, not playback.
2. A social network -- no user-to-user interaction, comments, or sharing.
3. A recommendation engine -- it displays existing ratings but does not generate personalized recommendations.
4. A content database -- it relies on TMDB and AniList APIs, no self-hosted media database.
5. A mobile app -- web-only (responsive, not native).
