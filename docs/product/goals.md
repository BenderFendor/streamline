# goals.md -- What the product should do

Streamline is an entertainment tracker that helps users track movies, TV shows, and anime in one centralized dashboard.

## Core goals

1. Browse trending and popular movies, TV shows, and anime.
2. Search and filter media by genre, category, format, and status.
3. View detailed information for any media item (cast, crew, ratings, trailers).
4. Maintain a personal watchlist with progress tracking.
5. AMOLED dark theme with cinematic UI (custom cursor, floating posters, tilt cards).
6. Responsive design for desktop and mobile.

## Planned but not yet implemented

1. User authentication (NextAuth with GitHub provider -- route exists, package missing).
2. Persistent watchlist with database (Prisma + PostgreSQL or Supabase).
3. Book tracking (OpenLibrary API integration).
4. Data export (Markdown, JSON).
5. Rating and review system.
6. State management cache layer (SWR/React Query).
